Executive Summary

One self-contained HTML file capturing the entire Pigment editorial visual system.

The starter renders the brand bar, cover, this executive-summary card, a factsheet grid, callouts, numeric tables, and footer in a single page - so any pattern can be lifted directly into a new artefact.

Every colour resolves from a CSS-custom-property token block at the top of the stylesheet; brand refresh edits live in one place. Typography pairs DM Sans + Crimson Pro + JetBrains Mono, all free Google Fonts, with the Alternative-g glyph enabled by default.

Drop the file into a browser - it renders on-brand immediately, no build step, no preprocessing.

01 · Patterns

Beyond the cover, the same tokens drive every component.

Callouts, tables, cards, and section banners all reference the same CSS custom properties. Edit one token, every consumer updates. Below: a few of the supporting patterns the starter demonstrates.

Callout themes

Info callout
Background uses Cobalt Soft (#B4D7FF), accent border uses Cobalt Bold (#152D80). Pair each Soft with its matching Bold for callouts.
Positive callout
Validated results / success framing. Background Emerald Soft (#E8F5CE), accent Emerald Bold (#1D683F).
Warn callout
Attention / caveat. Background Ochre Soft (#FBEBB1), accent Ochre Bold (#E9A800). Use sparingly - one or two per section, never stacked.

Numeric tables

Data cells use JetBrains Mono with tabular numerals so columns align. Headers stay in DM Sans even when right-aligned (the th.num class only flips alignment, not the font).
