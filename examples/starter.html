Section heading in Crimson Pro

Body text sits in DM Sans Regular at 1.5 line-height. The Alternative-g glyph is enabled via font-feature-settings: "ss01" on - look at any lowercase "g" to see the brand-correct loop shape. Editorial whitespace and a constrained reading width (around 70ch) make paragraphs readable; resist stretching prose to the full page width.

Sub-section in Crimson Pro Medium

Italic emphasis inside headings uses <em> tags - the CSS keys on the element name, so <i> won't get the same treatment. Inside an already-serif h2, the <em> stays Crimson Pro italic for typographic continuity.

Info callout

Background uses Cobalt Soft (#B4D7FF), accent border uses Cobalt Bold (#152D80). Pair each Soft with its matching Bold for callouts.

Positive callout

Validated results / success framing. Background Emerald Soft (#E8F5CE), accent Emerald Bold (#1D683F).

Warn callout

Attention / caveat. Background Ochre Soft (#FBEBB1), accent Ochre Bold (#E9A800). Use sparingly - one or two per page section, never stacked.

Numeric table demonstration

Data cells use JetBrains Mono with tabular numerals so columns align. Headers stay in DM Sans even when right-aligned (the th.num class only flips alignment, not the font).

Elevated card surface

Cards use the brand --shadow recipe - two soft Carbon-tinted layers, never neutral grey. The shadow gives subtle elevation without breaking the editorial flatness of the underlying paper.
